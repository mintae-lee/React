# React App

프로젝트 소스 [Create React App](https://github.com/facebook/create-react-app).


## 로컬환경

프로젝트 폴더에서 아래를 실행

### 로컬서버 시작
`npm start`
FYI [http://localhost:3000](http://localhost:3000)


### 로컬 테스트
`npm test`
FYI [running tests](https://facebook.github.io/create-react-app/docs/running-tests)


### 빌드
`npm run build`

결과파일 폴더 `build`
